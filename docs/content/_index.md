ULinkGame 是构建在 ULinkRPC 与 ULinkActor 之上的游戏会话基础设施层。它选择轻量、进程内、游戏友好的 actor/mailbox 模型，是因为 Orleans 风格的分布式 actor 平台对本框架的目标场景过重、不够贴近游戏服务端运行方式。
