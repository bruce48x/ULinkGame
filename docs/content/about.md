ULinkGame is a game-session infrastructure layer built on ULinkRPC and ULinkActor. It exists because Orleans-style distributed actor hosting is too heavy and too enterprise-oriented for the game server workflows this project targets.

This site is built with Hugo and is intended for user-facing articles, release notes, and design explanations. Repository-internal contribution notes stay in `CONTRIBUTING.md`; package-specific implementation notes stay next to the package that owns them.
